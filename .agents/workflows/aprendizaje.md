# /aprendizaje

Este comando automatiza la creación de una nueva nota de aprendizaje en el proyecto.

## Pasos a seguir:

1. Lee las instrucciones detalladas en la skill: [aprendizaje](file:///c:/Proyectos/rpr/.agents/skills/aprendizaje/SKILL.md).
2. Lista el contenido de `c:\Proyectos\rpr\docs\01_aprendizaje` para determinar el siguiente número correlativo `XX`.
3. Solicita o identifica el problema, la solución, las alternativas y la fecha.
4. Crea el archivo con el formato `XX_aprendizaje_descripcion_problema.md`.
5. Asegúrate de incluir la introducción corta requerida.
